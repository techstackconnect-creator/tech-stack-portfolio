# STUDIO. — Premium Freelance Portfolio

Cinematic MERN portfolio. Frontend: React + Vite + Tailwind v3 + Framer Motion + GSAP + Three.js. Backend: Express + MongoDB.

> Note: This project no longer runs inside the Lovable preview. Export to GitHub and run locally / deploy to Vercel + Render.

## Run locally

```bash
# frontend
bun install   # or npm install
bun run dev

# backend
cd server
npm install
cp .env.example .env   # add MONGODB_URI
npm run dev
```

## Deploy

- Frontend → Vercel (`vercel.json` SPA rewrite included)
- Backend → Render / Railway
- DB → MongoDB Atlas
